### Run App:

```bash
yarn run start
```

### Run Tests:

```bash
yarn run test
```

### Run Coverage:

```bash
yarn run coverage
```

### Run Prettier:

```bash
yarn run prettify
```

### Run Linting:

```bash
yarn run lint
```

### Setup a Mock DB with some fake data:

```bash
yarn run mock-db
```
